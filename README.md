# Sales Performance Dashboard (Power BI)

An interactive Power BI dashboard designed to analyze sales performance across time, products, and geography using Excel data and DAX measures.

---

## Objective
To provide business stakeholders with a clear view of sales trends, profitability, top-performing products, and city-wise performance through an interactive dashboard.

---

## Tools & Technologies
- Power BI Desktop
- Excel
- DAX
- SQL (analysis queries)

---

## Key KPIs
- Total Sales
- Total Profit
- Total Quantity Sold
- Profit Margin

---

## Dashboard Features
- Monthly Sales Trend analysis
- Top 10 Products by Sales
- City-wise Sales Distribution (Map)
- Interactive Year slicer
- Clean and business-friendly layout
